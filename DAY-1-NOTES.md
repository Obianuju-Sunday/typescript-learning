# TypeScript Learning - Day 1

TypeScript is JavaScript with "types" - it's like adding labels to your code so you know what kind of data goes where.

In regular JavaScript, you can put any type of data anywhere and it won't complain until something breaks.

In TypeScript, you tell it "this variable should be a number" or "this function needs a string" and if you try to use the wrong type, it tells you BEFORE you run the code. Quite fascinating!


## What I learned today:
- TypeScript adds types to JavaScript
- Basic types: string, number, boolean, array, object, any, function
- Types define object shapes and what properties they have
- tsc compiles .ts to .js
- The browser can only run JavaScript, so we need to compile TypeScript to JavaScript before running it in the browser
